MyWords para Common Utilities

¡Gracias por descargar MyWords. Este módulo te permitirá crear una sección para noticias, artículo o un blog en tu sitio web de manera rápida y sencilla, para que te concentres en escribir y te olvides de configuraciones y las complicaciones que presentan módulos similares.

Requerimientos

  • Esta versión ha sido ampliamente probada en XOOPS 2.5. Te recomendamos utilizar la versión mas reciente de esta serie (actualmente 2.5.7)
  • Common Utilities 2.2

Instalación

Importante: antes de instalar MyWords es necesario tener instalado rmcommon de lo contrario el módulo no podrá instalarse correctamente.

  1. Extrae el contenido del archivo comprimido en tu disco duro.
  2. Sube la carpeta «mywords» al directorio «modules» de tu instalación XOOPS.
  3. Ingresa al panel de control de XOOPS, luego a la administración de módulos de Common Utilities e instala MyWords como un módulo normal.
  4. ¡Listo! MyWords esta instalado y listo para usarse.

Obtener MyWords

Puedes obtener la versión mas reciente de MyWords directamente desde GitHub.

También puedes obtener la versión de desarrollo haciendo un fork del repositorio en GitHub.

Importante: La versión en desarrollo no debe ser usada en sitios en producción.

Ayuda y reporte de errores

Para reportar errores y problemas con el módulo, por favor utiliza la herramienta de GitHub. Es la mejor forma, y la única que reviso regularmente.
